Documentation Web Pelanggaran SMKN 4

1.Home Page

homepage adalah halaman awal yang terdapat pada web ini, dan tempilan nya berupa data dari para pewlanggar mulai dari nama pelanggar,kelas,jenis pelanggarannya,tanggal melakukan pelanggarannya hingga siapa yang telah menulis pelanggar tersebut. disana juga di tampilkan icon (+) di gunakan untuk shortcut ke page tambah pelanggar

2.Login Page

Halaman login digunakan untuk menuju adminpage ini merupakan sebuah validasi agar tidak sembarangan orang bisa menambahkan data pelanggar,untuk mengakses web ini kalian bisa ke icon tambah (+) pada homepage atau langsung memasukan url /masuk/

3.Dashboard

Dashboard page adalah halaman utama atau pusat kendali pada sebuah aplikasi atau sistem informasi yang menampilkan informasi penting dan statistik terkait kinerja dan performa sistem tersebut. Pada konteks yang disebutkan, dashboard page menampilkan data siswa, data pelanggar, serta total petugas dan total pelanggaran. Fungsi dari dashboard page tersebut adalah untuk memberikan gambaran yang cepat dan mudah dipahami tentang situasi dan kondisi terkini terkait data siswa, data pelanggar, dan performa petugas dalam menangani pelanggaran. Dengan informasi yang tersaji pada dashboard page, pengguna sistem dapat dengan mudah memantau dan mengevaluasi kinerja sistem dalam menangani pelanggaran serta dapat menentukan aksi atau strategi yang tepat untuk meningkatkan performa sistem. page ini dapat di akses di adminpage lalu pada sidebarnya terdapat dashboard
